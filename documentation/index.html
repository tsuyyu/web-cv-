Template Structure

Full list of plugins and libraries is available in Credits section of the documentation.

Stylesheets

  • css/vendors.css
    Bundle of libraries styles.
  • css/style.css
    Contains template styles with table of contents.

Scripts

  • js/vendors.js
    Bundle of libraries scripts.
  • js/main.js
    Contains template functionality with table of contents.

Images

  • img/
    Images of the template sections. There are grey placeholders which you should replace with your own images.

Fonts

Horizontal and vertical spacing

Control horizontal and vertical spacing of an element using the mx-{size} utilities.

Responsive

To control the margin of an element at a specific breakpoint, add a {screen}: prefix to any existing margin utility. For example, adding the class md:mt-20 to an element would apply the mt-20 utility at medium screen sizes and below.

            
              <div class="mt-28 md:mt-20 sm:mt-12"></div>

AJAX Transitions

The template is powered by AJAX transitions thanks to Barba.js plugin.

WARNING

You need to use any web-server to make AJAX transitions work.

AJAX link attribute

For the transitions to work properly, all links that lead to other pages of the site must have the data-barba attribute.

            
              <a data-barba href="anotherPage.html"></a>

Markup

Everything outside Barba container <main data-barba="container"> will remain unchanged when navigating the website. 

            
              <body data-barba="wrapper">
                ...
                  <div class="barba-container" data-barba="container">
                    ...
                  </div>
                ...
              </body>

To turn off AJAX transitions, you can:

  • Remove attribute data-barba="wrapper" from the body container.

  • Unwrap everything from this container since it doesn't create any styling in the document.

  • Set ajax.enabled property to false, in main.js file.

Lazy Loading

All images (including backgrounds) are loaded in a deferred way to speed up pages loading.

Images

For images in <img> tag, use data-src attribute with .js-lazy class.

            
              <img class="js-lazy" src="#" data-src="lazy.jpg" alt="A lazy image">

Background images

For background images, use data-bg attribute with .js-lazy class.

            
              <div class="bg-image js-lazy" data-bg="lazy.jpg"></div>

Reveal Animations

Single Animation

Template comes with several build in section reveal animations. To add or change an elementâ€™s reveal animation, you need to change [data-anim] attribute. 

            
              <div data-anim="slide-up"></div>
              <div data-anim="slide-down"></div>
              <div data-anim="slide-left"></div>
              <div data-anim="slide-right"></div>
              <div data-anim="fade"></div>
              <div data-anim="img-right"></div>

If animation doesn't work properly try to wrap the element you want to animate.

            
              <div data-anim="slide-up">
                Element You want to animate
              </div>

Animation Container

If you want to animate multiple elemets at once, wrap them in data-anim-wrap container and use data-anim-child insted of data-anim attribute. 

            
              <div data-anim-wrap>
                <div data-anim-child="slide-up"></div>
                <div data-anim-child="slide-up"></div>
                <div data-anim-child="slide-up"></div>
              </div>

Delay

By adding additional delay- property, you can change animation delay. Every next value adds 100 milliseconds. 

            
              <div data-anim="slide-up delay-1"></div>
              <div data-anim="slide-up delay-2"></div>
              <div data-anim="slide-up delay-3"></div>
              ...
              <div data-anim="slide-up delay-20"></div>

              <div data-anim-child="slide-up delay-1"></div>
              <div data-anim-child="slide-up delay-2"></div>
              <div data-anim-child="slide-up delay-3"></div>
              ...
              <div data-anim-child="slide-up delay-20"></div>

AJAX Contact form

The template comes with simple PHP mail script.

Open contact.php file with any text editor and change $toEmail variable to your own e-mail address.

Please note:

  • Please note that you need PHP to be enabled on your hosting with mail() allowed function.
  • Email could get flagged as spam.
          
            <?php

              if ($_POST) {
                $toEmail = 'example@gmail.com';
                $values = [];
                
                if ($_POST['name']) {
                  $subject = 'Contact Request From ' . htmlspecialchars($_POST['name']);
                } else {
                  $subject = 'Contact Request';
                }

                foreach ($_POST as $propName => $propValue) {
                  if ($propName !== 'message') {
                    $values[] = 'Visitor ' . htmlspecialchars($propName) . ': ' . htmlspecialchars($propValue);
                  }
                }

                if ($_POST['message']) {
                  $values[] = '<br>Visitor message:<br><br>' . htmlspecialchars($_POST['message']);
                }

                $body = implode('<br>', $values);
                $headers = "MIME-Version: 1.0" . "\r\n";
                $headers .= "Content-Type:text/html;charset=UTF-8" . "\r\n";
                
                if ($_POST['name'] || $_POST['email']) {
                  $headers .= "From: ";
                }

                if ($_POST['name']) {
                  $headers .= htmlspecialchars($_POST['name']);
                }

                if ($_POST['email']) {
                  $headers .= "<" .htmlspecialchars($_POST['email']). ">";
                }

                $headers .= "\r\n";

                mail($toEmail, $subject, $body, $headers);
              }

            ?>
